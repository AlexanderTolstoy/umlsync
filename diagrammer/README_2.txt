Diagrammer
============

Diagrammer is a lightweight JavaScript based framwork which
allow to create and manage diagrams. It is provide the default
configurable list of elements and mehanism for creation a new
element or extend an existing elements.
Lazy load mehanism allow to minimize the client part and
easily load required diagram types.

It is being developed for UML diagrams managment, but
easily extensibility allow to extend the diagram types
and functionality.

It currently has support for SVN and Git repositories.

Information on usage and installation can be found on
http://www.umlsync.org/docs/

General information on the project is available on
http://www.umlsync.org/
